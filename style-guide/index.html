FONTS

The primary typefaces for Shelley Röstlund are Rubik, Lexend, Rocksalt and La Belle Aurore. Rubik is suitable for headline applications and can be used in Medium or Bold.

Lexend should be used for body copy and captions and can be used in Light, Regular or Medium. Rocksalt is the designer font and can be used to pull out or highlight important points.

The brand can be supported by the handscript font La Belle Aurore. This font is suitable for use in quotes, tips or hits if appropriate. It should be used sparingly in single sentences as a design feature and should not be used in captials.
